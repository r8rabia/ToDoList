-To-Do List Web Application
This project is a basic To-Do List web application built using HTML, CSS, and JavaScript. It allows users to create and manage tasks through a simple and interactive interface. The application demonstrates fundamental JavaScript concepts such as DOM manipulation, event handling, array operations, and browser local storage.
The goal of this project was to practice front-end development fundamentals while building a functional task management tool.

- Features
Add Tasks – Users can add new tasks to the list.
Search Tasks – Tasks can be filtered using the search functionality.
Remove Tasks – Individual tasks can be deleted.
Clear All Tasks – Users can reset the entire task list.
Persistent Storage – Tasks are stored using localStorage, so they remain saved even after refreshing the page.

 -Technologies Used
HTML – Structure of the web page
CSS – Styling and layout
JavaScript – Logic, DOM manipulation, and functionality
Local Storage API – Saving tasks in the browser

- How It Works
The user enters a task in the input field.
When Add Task is clicked, the task is stored in a JavaScript array.
The task list is dynamically rendered in the browser.
Tasks are saved to localStorage so they persist after refresh.
Users can search tasks or delete them individually.
